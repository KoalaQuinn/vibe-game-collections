# 🔯 修真放置

纯 JavaScript 编写的修真题材放置挂机游戏，开源免费，开箱即用。

## 玩法介绍

- **在线吐纳**：打开网页自动涨灵气，提升境界
- **离线灵气**：关闭网页后丹田自动积累灵气，下次打开手动领取，最多存 24 小时
- **打通经脉**：消耗修为提升属性，打开丹田奇经八脉
- **修炼功法**：提升内功、法术、身法、防御
- **秘境探险**：挂机打怪，收集灵草、魔晶、妖骨
- **炼丹**：用灵草炼制聚气丹，恢复气血内力

## 境界列表

- 炼气 → 筑基 → 金丹 → 元婴 → 化神
- 炼虚 → 合体 → 大乘 → 渡劫 → 飞升

每 10 级提升一个大境界。

## 如何运行

直接用浏览器打开 `index.html` 就能玩，不需要任何后端。

或者启动一个本地服务器：

```bash
# Python 3
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 存档

所有数据存在浏览器本地存储（localStorage），清理浏览器缓存会丢失，重要进度请自行备份。
