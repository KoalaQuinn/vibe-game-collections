# ⚔️ 三国立志传：草莽英雄

太阁风格三国文字模拟游戏，你是乱世中的小人物，从流民慢慢成长。开放玩法，没有固定结局，活下去就是胜利。

## 游戏特色

- **小人物成长**：从破败草庐开始，慢慢升级领地，成为一方豪强
- **多线养成**：种田修炼、当官经商、结交豪杰、带兵打仗，自由选择
- **事件可配置**：所有事件可以用 Excel 配置，导出 JSON 就能用，不需要改代码
- **纯前端网页**：打开就能玩，数据存在本地，不需要后端

## 核心玩法

### 回合

一个月 = 一回合，每个月有 5 点行动力，每个行动消耗 1 点，用完点「结束本月」结算。

### 属性

四个基础属性，都可以升级：
| 属性 | 作用 |
|------|------|
| 武力 | 开垦、战斗 |
| 智力 | 政务、事件判定 |
| 魅力 | 招募、结交 |
| 统率 | 带兵、战斗伤害 |

### 资源

| 资源 | 作用 |
|------|------|
| 金钱 | 升级、买卖请客 |
| 粮食 | 每月消耗，吃完饿死 |
| 人口 | 增加产出 |
| 士兵 | 打仗，每个每月吃 1 粮食 |

### 地点

| Tab | 内容 |
|-----|------|
| 我的领地 | 属性、资源、日常行动 |
| 州郡街市 | 官署接任务、酒馆刷 NPC 和事件、市集买卖 |
| 天下大势 | 历史日志 + NPC 羁绊列表 |

### 事件系统

所有事件都可以用 Excel 配置，结构如下：

| 列名 | 说明 |
|------|------|
| `event_id` | 唯一ID |
| `trigger_type` | `random`/`date`/`condition` |
| `trigger_date` | `date` 类型填 `YYYY-MM` |
| `title` | 事件标题 |
| `description` | 事件描述 |
| `opt1_text` | 选项1文字 |
| `opt1_attr` | 需要检定的属性（空不需要） |
| `opt1_diff` | 检定难度 1-100 |
| `opt1_success_text` | 成功文字 |
| `opt1_success_reward` | JSON 奖励，比如 `{"money": 150, "exp_force": 5}` |
| `opt1_fail_text` | 失败文字 |
| `opt1_fail_reward` | JSON 惩罚 |
| `opt2_text` ... | 最多支持3个选项 |

导出 CSV → 转 JSON 就能导入游戏。

## 怎么运行

直接浏览器打开 `index.html` 就能玩。

或者本地启动：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 存档

所有数据存在浏览器 localStorage，清缓存会丢失，重要存档请备份。
