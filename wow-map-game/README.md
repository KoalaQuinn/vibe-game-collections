# 魔兽世界艾泽拉斯大地图 - 网页探索游戏

一个纯前端 2D 魔兽世界大地图探索游戏，支持拖拽缩放，点击区域查看信息。

## 功能特性

- ✅ 完整艾泽拉斯大地图显示
- ✅ 支持鼠标拖拽移动地图
- ✅ 支持滚轮缩放
- ✅ 预定义了主要主城和区域的可点击热点
- ✅ 点击弹出区域介绍和可交互动作
- ✅ 移动端触摸支持
- ✅ 魔兽世界风格UI

## 已添加的区域

- **联盟**：暴风城、铁炉堡、达纳苏斯
- **部落**：奥格瑞玛、雷霆崖、幽暗城
- **中立/其他**：藏宝海湾、希利苏斯

## 运行方式

直接用浏览器打开 `index.html` 即可运行：

```bash
# 如果你有 Python，可以启动一个本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 添加新区域

在 `game.js` 中的 `WORLD_REGIONS` 对象添加新区域：

```javascript
"your-region": {
    name: "区域名称",
    description: "区域描述...",
    faction: "联盟/部落/中立/双方",
    level: "等级范围",
    location: {x: X坐标, y: Y坐标, radius: 点击半径},
    actions: [
        {text: "按钮文字", action: "动作类型"},
        ...
    ]
}
```

坐标说明：x, y 是相对于原图（左上角为 0）的坐标，可以根据实际地图调整。
