# vibe-game-collection

🎮 一组轻量级网页小游戏集合，纯前端实现，零依赖，直接可以玩。

## 游戏列表

| 游戏 | 描述 | 链接 |
|------|------|------|
| **艾泽拉斯放置** | 魔兽世界风格放置RPG，类似战斗无止境 (Battle Without End)，自动挂机刷怪打装备 | [wow-azoth-idle](./wow-azoth-idle/) |

## 运行

每个游戏都是独立的纯静态网页，直接打开 `index.html` 就能玩，或者用任何静态文件服务器运行：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000/游戏目录/
```
