# 武侠放置 - 单机放置类武侠游戏

## 项目概述
- 类型：武侠放置 / 挂机养成 单机游戏
- 开发工具：Godot 4.x + godot-mcp
- 目标：先做出可玩版本自己玩，使用程序化图形，不需要美术资源

## 核心设计思路

### 核心玩法
1. **挂机修炼**：离线也能获得修为，上线领取
2. **经脉丹田**：打通经脉，拓展丹田，提升内力上限
3. **武学修炼**：收集武功秘籍，升级招式心法
4. **江湖闯荡**：自动打怪，收集材料和装备
5. **奇遇事件**：随机事件获得天赋、秘籍、资源

### 核心属性
- 修为：升级等级用
- 内力：发动招式需要，修炼自动回复
- 气血：生命值，被打会掉，回血需要时间或药品
- 攻击/防御：基础战斗属性
- 悟性：影响修炼速度，奇遇概率

### 养成线
1. 人物等级提升
2. 经脉打通顺序选择
3. 武学搭配组合（内功+外功+轻功）
4. 装备打造/强化
5. 天赋树解锁

### 界面设计（极简第一版）
- 主界面：属性展示 + 修炼按钮 + 闯荡按钮
- 经脉界面：可视化丹田经脉，点击打通
- 武学界面：秘籍列表，修炼升级
- 背包/装备界面：简单列表展示

### 美术方案
- 所有图形使用Godot着色器和形状节点程序化生成
- 丹田：圆形渐变，打通的经脉用线条连接
- 人物状态：简单色块表示
- UI：系统默认风格，够用就行
