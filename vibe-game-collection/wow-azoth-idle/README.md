# 艾泽拉斯放置 - 魔兽世界风格放置RPG

类似 **Battle Without End (战斗无止境) 玩法的放置点击游戏，魔兽世界世界观。

## 玩法特点

- ✅ 自动战斗挂机，完全离线也打
- ✅ 击杀怪物获得经验和金币
- ✅ 随机掉落装备，不同品质
- ✅ 装备系统，6个装备槽
- ✅ 金币购买属性提升
- ✅ 本地存储进度
- ✅ 魔兽世界怪物（从1级开始挑战直到60级

## 已实现系统

1. **战斗系统**
   - 手动点击攻击
   - 自动战斗开关，开了就自动打
   - 怪物会反击
   - 死亡复活机制

2. **装备系统**
   - 6个装备部位：头/颈/肩/胸/武器/戒指
   - 装备品质：粗糙 → 普通 → 优秀 → 稀有 → 史诗 → 传说
   - 不同品质不同颜色边框和文字
   - 鼠标悬停显示装备属性
   - 点击装备装备，满了卖掉

3. **成长系统**
   - 升级自动涨属性
   - 可以用金币训练提升攻击/生命

4. **怪物**
   目前内置了10种魔兽世界怪物，从史莱姆到上古之神，自动匹配玩家等级生成

## 运行方式

```bash
cd /root/.openclaw/workspace/wow-idle-game
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 扩展

- 添加更多怪物到 `MONSTERS` 数组
- 添加更多装备模板到 `EQUIPMENT_TEMPLATES`
- 修改 `autoAttackInterval` 调整自动攻击速度

## 技术

纯原生JS + HTML5 + CSS，无依赖，直接就能跑。
