## 🚀 如何添加新游戏

1. 在项目根目录新建你的游戏文件夹，放入游戏文件（确保有 `index.html`）
2. 修改根目录的 [`index.html`](./index.html)，在 `.games-grid` 区域添加一张新的游戏卡片
3. 提交并推送到 GitHub，GitHub Pages 会自动更新

## 🎨 首页效果

首页采用渐变深色背景 + 卡片式设计，响应式布局适配电脑和手机访问。每个游戏卡片包含：
- 游戏名称
- 简短描述
- 标签（类型、技术栈）
- 点击直接跳转游玩
