# 即时钓鱼游戏设计文档

## 需求概述
- 即时操作类钓鱼游戏，偏动作感，高画面表现力
- 纯网页，移动端竖屏适配，单页应用
- 简约扁平化现代风格

## 核心流程
1. **长按蓄力** - 用户长按按钮，蓄力条增长，松手即抛竿
2. **抛竿动画** - Canvas 动画：鱼竿弯曲 → 鱼线飞出 → 浮漂落水
3. **等待咬钩** - 浮漂随机晃动，震动反馈，等待下沉瞬间
4. **即时拉杆** - 正确时机点击拉杆，成功则钓起鱼，失败则逃走
5. **结算收获** - 记录到图鉴，卖鱼得金币，更新经验

## 系统设计

### 1. 鱼类系统
每条鱼属性：
```json
{
  "id": "crucian",
  "name": "鲫鱼",
  "icon": "🐟",
  "minWeight": 100,
  "maxWeight": 500,
  "rarity": "common", // common(1) / rare(2) / epic(3) / legendary(5)
  "price": 2, // 每100g价格
  "unlockAt": "pond", // 哪个钓点解锁
  "difficulty": 1 // 拉杆难度，越高越难
}
```

### 2. 钓点
- pond 池塘（初始）
- river 河流（1000金）
- lake 湖泊（5000金）
- sea 大海（20000金）

每个钓点有不同的鱼列表，不同概率出稀有鱼

### 3. 成长系统
- 鱼竿：等级提升 → 增加力量（能钓更大鱼）、增加命中率
  1. 竹杆 → 2. 玻璃钢 → 3. 碳素 → 4. 神器
- 鱼饵：玩家选择装备，不同鱼饵吸引不同鱼类，增加对应鱼概率
  消耗品，每次钓鱼消耗一个

### 4. 图鉴系统
- localStorage 记录已钓到的鱼，最大重量，钓到次数
- 图鉴界面显示收集进度

### 5. 奇遇系统
- 5% 概率钓到非鱼：
  - 木箱：+50~200金
  - 宝箱：+200~1000金
  - 垃圾（旧靴子/破瓶子）：+1~10金
  - 水怪BOSS：特殊难度，成功给1000+金

## 技术实现
- 主场景：Canvas 2D 绘制动画
- UI：HTML + CSS 实现响应式
- 存档：LocalStorage
- 所有代码在单个 index.html + game.js 方便部署
